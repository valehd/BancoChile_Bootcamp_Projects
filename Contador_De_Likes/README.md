
# ❤️ Feed Interactivo --> contador-de-likes

Aplicación web desarrollada con HTML, CSS y JavaScript que simula un feed de publicaciones con sistema de "Me gusta".

## 🚀 Descripción

Este proyecto permite interactuar con distintas publicaciones mediante botones de "Like". Cada publicación mantiene su propio contador, el cual se actualiza dinámicamente utilizando JavaScript.

## ✨ Funcionalidades

* Visualización de publicaciones en formato de feed.
* Contador independiente de likes para cada publicación.
* Actualización dinámica del contenido mediante manipulación del DOM.
* Interacción mediante eventos de clic.

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

## 📚 Conceptos practicados

* Selección de elementos con `querySelectorAll`.
* Recorrido de colecciones mediante bucles.
* Manejo de eventos con `addEventListener`.
* Actualización dinámica de texto con `textContent`.
* Uso de variables y contadores independientes.
