# Batalla-musical

Aplicación web interactiva desarrollada con HTML, CSS y JavaScript que permite filtrar géneros musicales y votar por bandas participantes en una batalla musical.
## 🚀 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## 📋 Descripción

Este proyecto consiste en el desarrollo de una página web que integra estructura semántica, estilos personalizados y funcionalidades dinámicas mediante JavaScript.

Entre las características implementadas se encuentran:

- Estructura semántica con HTML5.
- Diseño responsivo.
- Manipulación del DOM.
- Eventos de interacción con el usuario.
- Organización de archivos y recursos del proyecto.
