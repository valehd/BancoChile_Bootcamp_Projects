# 👤 Perfil Social

Página de perfil desarrollada con HTML y CSS que simula una red social profesional.

## 🚀 Descripción

Este proyecto consiste en la creación de una interfaz de perfil de usuario inspirada en plataformas de networking profesional. Incluye información personal, educación, solicitudes de conexión y una lista de contactos.

## ✨ Características

* Barra de navegación superior.
* Tarjeta de perfil con información personal.
* Sección de educación.
* Solicitudes de conexión.
* Lista de conexiones.
* Uso de iconografía e imágenes de perfil.
* Diseño organizado mediante Flexbox.

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3

## 📚 Conceptos practicados

* Estructuración semántica con HTML.
* Maquetación con Flexbox.
* Diseño de tarjetas (Cards).
* Uso de imágenes e iconos.
* Organización de layouts responsivos.
