# core_tienda_productos

Resumen de Relaciones
Categorías 1 : N Productos (Una categoría tiene muchos productos; un producto pertenece a una sola categoría)

Tiendas 1 : N Inventarios (Una tienda tiene muchos registros de inventario)

Productos 1 : N Inventarios (Un producto puede estar en los inventarios de muchas tiendas)

Proveedores 1 : N Pedidos (Un proveedor recibe muchos pedidos)

Tiendas 1 : N Pedidos (Una tienda genera muchos pedidos)
