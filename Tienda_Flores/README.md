# 🌸 FlorMania | Interactive Flower Shop

## Overview

FlorMania is a responsive web application developed to demonstrate fundamental front-end development skills using HTML, CSS, and JavaScript. The project focuses on DOM manipulation, event handling, and dynamic user interactions within an online flower shop interface.

## Features

### User Welcome Interaction

Displays a personalized welcome message when the user clicks the **"Iniciar Sesión"** button.

### Dynamic Navigation Content

Updates the navigation text from **"Envía Flores"** to **"Envía Bouquetes"** when hovering over the menu item and restores the original text when the cursor leaves.

### Interactive Purchase Buttons

Allows users to interact with product cards by hiding the selected **"Comprar"** button after clicking it, simulating a purchase action.
