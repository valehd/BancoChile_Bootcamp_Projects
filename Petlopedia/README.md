# 🐾 Petlopedia

Aplicación web desarrollada con HTML, CSS y JavaScript que simula una enciclopedia de mascotas con funcionalidades interactivas.

## 🚀 Descripción

Petlopedia presenta información sobre distintas mascotas y permite a los usuarios interactuar con la página mediante botones de inicio de sesión, me gusta y administración de contenido.

## ✨ Funcionalidades

* Cambiar el botón de "Iniciar sesión" a "Cerrar sesión".
* Incrementar el contador de "Me gusta" para cada mascota.
* Mostrar alertas al registrar un "Me gusta".
* Ocultar el botón "Agregar definición" al hacer clic.

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

## 📚 Conceptos practicados

* Selección de elementos con `getElementById`.
* Manejo de eventos con `addEventListener`.
* Modificación dinámica de texto (`innerText`).
* Manipulación de estilos mediante JavaScript.
* Uso de variables y contadores.
